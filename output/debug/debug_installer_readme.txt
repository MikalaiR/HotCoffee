Attention to all HotCoffee users!
If you have a problem with the package installer, then:
1. Download and run the file debug_installer.exe
2. Click "Click me and wait" or "Нажми меня и жди" (after the completion debug.log will appear)
3. send debug.log with all its contents to im.hotcoffee@gmail.com via email


Вниманию пользователей сборки HotCoffee!
Если у вас есть проблемы с инсталятором сборки:
1. Скачиваем и запускаем файл debug_installer.exe
2. Жмем кнопку "Нажми меня и жди" (после завершения процесса рядом появится файл debug.log)
3. Файл debug.log или все его содержимое отправляем на e-mail: im.hotcoffee@gmail.com
